# Release Notes - MVP 0.8.0

Date: 2026-02-23

## Summary
First full MVP release of the DnB Bass Synth VST3 with closed P0-P8 delivery scope.

## Included
1. Restriction-first domain constraint system (`ConstraintEngine` as single gate).
2. Dual-engine audio path (Sub + Character) with deterministic note behavior.
3. Canonical MVP styles: `Tech`, `Neuro`, `Dark`.
4. Complete MVP macro surface:
   - 10 macro controls (4 primary + 6 advanced)
   - separate style morph control
   - `macro.neuro_formant`
   - `macro.fm_metal`
   - `macro.roller_dynamics`
   - `macro.liquid_depth`
   - `macro.style_morph`
   - `macro.sub_punch`
   - `macro.fm_pressure`
   - `macro.cutoff_motion`
   - `macro.fold_bite`
   - `macro.table_drift`
   - `macro.smash_glue`
5. Preset/state save-load with migration support.
6. Stylized UI skin integration and DAW hotfixes for endpoint and visual overlap regressions.
7. CI/CD hardening with build/test gates and formatting/lint enforcement.
