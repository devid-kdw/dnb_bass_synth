## 4. Mandatory File Name Formula
Use this exact format:
`ui_<group>_<component>_<variant>_<state>_v<NNN>.png`

Rules:
1. lowercase snake_case only
2. ASCII only
3. no spaces
4. one semantic asset per file
5. version must be zero-padded integer (`v001`, `v002`, ...)

Allowed group values:
- `background`
- `panel`
- `macro`
- `style`
- `visual`
- `badge`
- `icon`

Allowed state values:
- `base`
- `idle`
- `hover`
- `active`
- `selected`
- `disabled`
- `on`

Regex (validation):
`^ui_(background|panel|macro|style|visual|badge|icon)_[a-z0-9_]+_[a-z0-9_]+_(base|idle|hover|active|selected|disabled|on)_v[0-9]{3}\.png$`

## 6. PNG Technical Requirements
1. Format: PNG, RGBA, 8-bit per channel
2. Color profile: sRGB
3. Alpha: straight alpha (non-premultiplied)
4. Artboard fit: no accidental transparent padding drift between states
5. Export scale: 1x mandatory

## 7. Reporting and Handoff (Mandatory)
Designer must deliver:
1. Manifest CSV:
   - `assets/ui/designer/p7/manifests/p7_ui_png_manifest.csv`
   - exactly 60 rows
   - columns:
     - `file_name`
     - `category`
     - `component_id`
     - `state`
     - `width_px`
     - `height_px`
     - `alpha_mode`
     - `checksum_sha256`
     - `notes`
2. Mapping note:
   - `assets/ui/designer/p7/manifests/p7_ui_control_asset_map.md`
   - maps each UI control to exact PNG file IDs
3. Handoff report:
   - `docs/qa/design_phase_handoffs/P7_design_handoff.md`
   - must include:
     - delivery summary
     - generated file count by category
     - missing/blocked items
     - known visual compromises
     - confirmation that naming formula is respected

## 8. Frontend Consumption Rules
Frontend agent must:
1. read `p7_ui_png_manifest.csv` and `p7_ui_control_asset_map.md`
2. consume only files that pass naming regex and are listed in manifest
3. copy/integrate approved assets into `assets/ui/runtime/p7/`
4. avoid hardcoded fallback paths outside contract
5. report all asset substitutions in frontend handoff

## 9. Change Control
Any change to asset count, naming formula, or state matrix requires:
1. update of this contract file
2. note in phase handoff
3. ADR only if change affects architecture or cross-agent interfaces
