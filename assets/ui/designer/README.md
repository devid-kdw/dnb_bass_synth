# UI Designer Asset Workspace

This directory stores visual design deliverables produced by the UI Graphic Designer agent.

## Structure
- `p7/source/`: editable design source files
- `p7/exports/png/`: exported PNG assets by category
- `p7/manifests/`: CSV and mapping documents
- `p7/previews/`: contact sheets and review snapshots
- `p7/archive/`: superseded asset versions

Frontend integrates approved assets into `assets/ui/runtime/p7/`.

Naming and count rules are defined in:
- `docs/spec/ui_graphic_asset_contract.md`
